## Home: Dashboard & Performance Overview

Segments allow banks to create specific customer/company groups for better targeting, analysis, and automation.

### Features:

* Visual Rule Builder – Define segments using a no-code rule engine.

* SQL Query Generation – Automatically converts segment rules into SQL queries.

* Scheduled Execution – Runs segmentation queries on a defined schedule.
